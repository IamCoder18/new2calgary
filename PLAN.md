## Technical Approach

### Option A: Static HTML Site (Recommended)
- **Tool:** Plain HTML/CSS/JS or Hugo/Astro
- **Host:** GitHub Pages → custom domain
- **Cost:** $0 (or $12/year for domain)
- **Time to build:** 4-8 hours

### Option B: Notion + Super
- **Tool:** Notion + Super.so
- **Cost:** $10/month
- **Easier to update**

### Option C: WordPress
- **Tool:** WordPress on cheap hosting
- **Cost:** ~$5/month
- **Most flexible**

### Recommendation: **Option A** (fastest, free, good for SEO)

## Monetization Strategy

### 1. Referral Links (Primary)
- Place honest reviews with embedded referral links
- Use short links (easier to track: bit.ly/new2calgary-wealthsimple)
- Disclose: "Uses affiliate links"

### 2. Display Ads
- Google AdSense (after domain age)
- Or: direct sponsor banners from local businesses

### 3. Featured Listings
- Local businesses can pay for "Featured" placement ($50-100/year)
